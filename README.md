# 🎵 Moodfy — Seu Gosto em Dados

> 🚧 **Status do Projeto:** 🛠️ Em Desenvolvimento / Fase Inicial

O **Moodboard Musical** é uma aplicação web que se conectará à API oficial do Spotify para transformar o histórico de reprodução e os favoritos do usuário em insights visuais sobre o seu comportamento e gosto musical. 

A ideia é ir além das listas tradicionais e criar uma experiência visual interativa baseada nos metadados de áudio do Spotify (como energia, valência e dançabilidade).

---

## 🎯 Escopo do Projeto (O que será construído)

### 🖥️ A Experiência do Usuário (UI/UX)
- [ ] **Módulo de Autenticação:** Tela de login integrada diretamente com o Spotify.
- [ ] **Mapa de Humor (Mood Map):** Gráfico 2D cruzando *Energia* vs. *Valência* das músicas ouvidas.
- [ ] **Linha do Tempo:** Histórico mensal dos artistas que dominaram os fones do usuário.
- [ ] **Fingerprint Sonoro:** Um gráfico de radar (*radar chart*) comparando os atributos do usuário com a média global.
- [ ] **Top Músicas com Contexto:** Listagem detalhada com as características técnicas de cada faixa.

### ⚙️ Desafios Técnicos & Arquitetura (Foco em Front-End)
- [ ] **Autenticação Segura:** Implementação do fluxo **OAuth 2.0 com PKCE** (Client-Side).
- [ ] **Consumo de API Avançado:** Gerenciamento de paginação, cache local e tratamento de *Rate Limiting*.
- [ ] **Visualização de Dados:** Integração de gráficos robustos usando **D3.js** ou **Recharts**.
- [ ] **Gerenciamento de Estado:** Arquitetura limpa usando **Zustand** ou Context API.
- [ ] **Segurança de Rotas:** Criação de *Route Guards* no React Router para proteger o dashboard.
